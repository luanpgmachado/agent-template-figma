# Templates Corporativos - Saude

Usar esta skill para desenhar templates reutilizaveis no Figma com linguagem institucional, modular e executiva.

## Objetivo
- Entregar templates prontos para duplicacao.
- Manter leitura rapida, hierarquia forte e grid rigido.
- Garantir aparencia institucional premium, sem visual de startup colorida.

## Tipos de template
- Capa institucional
- Dashboard executivo
- Dashboard operacional
- Dashboard com menu lateral
- Dashboard analitico
- Tela de visao geral
- Tela de ocupacao e leitos
- Tela de centro cirurgico
- Tela de DRE ou financeiro
- Tela com mapa de unidades

## Fluxo
1. Definir tipo de template, publico, contexto e modo visual.
2. Montar frame e grid antes de desenhar detalhes.
3. Distribuir os blocos principais nesta ordem: header, KPI, conteudo, filtros, tabelas ou rankings, rodape.
4. Transformar blocos recorrentes em componentes reutilizaveis.
5. Revisar alinhamento, contraste, densidade e leitura.

## Estrutura padrao
### Header
- Posicionar logo a esquerda.
- Posicionar elemento institucional a direita.
- Exibir titulo e subtitulo com hierarquia clara.

### KPI
- Usar cards no topo.
- Destacar numeros com escala forte.
- Sinalizar variacao ou status de forma discreta.

### Conteudo
- Combinar graficos de barras, linha e pizza ou rosca quando fizer sentido.
- Reservar area para tabela ou ranking.
- Incluir filtros quando a tela pedir comparacao recorrente.

### Rodape
- Exibir data de atualizacao.
- Inserir assinatura discreta.

## Componentes obrigatorios
- Header
- KPI Card
- Chart Card
- Table Card
- Filter Bar
- Sidebar, quando houver navegacao lateral
- Cover

## Direcao visual
- Manter visual limpo, sobrio e corporativo.
- Trabalhar com espacamento amplo, alinhamento preciso e blocos bem definidos.
- Usar cards com leve arredondamento.
- Equilibrar blocos grandes e pequenos sem quebrar o grid.
- Evitar ornamento inutil, excesso de cor e poluicao visual.

## Paleta
- Base clara com branco e off-white.
- Texto forte em grafite ou preto.
- Verde institucional como cor de destaque.
- Em dark mode, usar base preta ou grafite e o mesmo verde como acento.

## Tipografia
- Usar sans-serif de alta legibilidade.
- Dar destaque forte aos KPIs.
- Manter hierarquia visivel entre titulo, subtitulo, card e legenda.

## Regras de decisao
- Se parecer apresentacao promocional, reduzir cor e ornamento.
- Se a leitura nao funcionar em poucos segundos, simplificar a composicao.
- Se um bloco nao puder ser reaproveitado, refatorar como componente.
- Se o grid perder consistencia, corrigir estrutura antes de ajustar estilo.
